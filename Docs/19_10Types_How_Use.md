#### 🧩 목표
이 문서를 기반으로 아래 조건을 충족하는 **30초 숏폼 영상 시나리오 3가지**를 생성하세요.

3. **BGM 제안(Background Music)**: 감정선에 맞는 음악 장르나 리듬감 설명  
4. **핵심 기능을 감각적으로 설명**: 어려운 기술은 비유·시각적 은유로 표현

- 타겟: MZ세대 직장인

- 스타일 예시: Notion / Slack / Zapier / ClickUp / Midjourney 캠페인 톤 참고  
- 길이: 각 시나리오 30초 분량  
- 형식: 마크다운 표로 정리
