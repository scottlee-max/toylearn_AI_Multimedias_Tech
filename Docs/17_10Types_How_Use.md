## 🧩 1. 최적 프롬프트 유형 및 선택 이유

| 구분 | 선택 유형                         | 이유                                                                                        |
| -- | ----------------------------- | ----------------------------------------------------------------------------------------- |
| ①  | **역할 부여형 (Role-Playing)**     | ‘멀티미디어 콘텐츠 매니저’라는 전문 역할을 명시해야 실무적이고 현실적인 전략을 도출할 수 있음.                                    |
| ②  | **제약 조건형 (Constraint-Based)** | 각 플랫폼(유튜브, 릴스, 틱톡, 블로그)의 특성, 영상 길이 제한, 톤앤매너 등을 명확히 제시해야 결과물이 규격에 맞음.                      |
| ③  | **재구성/번역형 (Restructuring)**   | 하나의 15분 원본 영상 소스를 플랫폼별로 최적화된 형태로 ‘재가공(Repurposing)’해야 함. 즉, 기존 콘텐츠를 다른 형태로 변환하는 것이 핵심 과제. |

➡ **따라서 “역할 부여 + 제약 조건 + 재구성형 복합 프롬프트”** 조합이 최적입니다.
