## 1\. 최적의 프롬프트 유형 선택 및 이유 설명

### 선택 유형: 체인 오브 스루트 (Chain of Thought, CoT)

| 유형 | 난이도 | 주요 목적 | 적합한 작업 |
| :--- | :--- | :--- | :--- |
| **체인 오브 스루트 (CoT)** | ⭐⭐⭐ | **복잡한 추론**, **문제 해결** | 데이터 기반 원인 분석, 경영/전략적 의사 결정 |

### 선택 이유 (기업 리스크 해결 관점)

제시된 문제는 단순 정보 요약이나 콘텐츠 생성이 아닌, \*\*복합적인 데이터(CTR, 전환율, 이탈률)\*\*를 분석하고 \*\*광고 크리에이티브(할인 혜택)\*\*의 실패 원인을 추론하여 \*\*즉각적인 비즈니스 리스크(클라이언트의 예산 환불 및 계약 중단)\*\*를 해결하는 것이 핵심입니다.

1.  **원인 분석 능력 극대화:** 훈련 초점 자체가 \*\*[원인 분석]\*\*입니다. CoT는 AI에게 최종 결과(대응 방안)만 요구하는 것이 아니라, 결과에 도달하는 \*\*논리적 추론 과정(단계별 분석)\*\*을 강제합니다. 이는 '할인 혜택 영상이 왜 실패했는가'라는 복잡한 인과관계를 놓치지 않고, 데이터와 크리에이티브를 연결하는 심층적인 진단을 가능하게 합니다.
2.  **리스크 관리:** 기업 리스크 전문가의 관점에서, 클라이언트에게 보고할 때는 '왜(Why)' 실패했는지에 대한 명확한 논리가 필요합니다. CoT를 통해 도출된 단계별 분석은 보고서의 설득력과 신뢰도를 높여, 클라이언트의 우려를 효과적으로 해소하고 예산 환불과 같은 극단적인 리스크를 방어하는 데 유리합니다.
3.  **실행 가능성 확보:** CoT는 추론 과정의 마지막 단계에서 필연적으로 '해결책'을 요구하게 되므로, 원인 진단에만 머무르지 않고 문제 해결에 직결되는 **구체적인 수정(Re-edit) 전략**까지 도출할 수 있습니다.

## 2\. 문제 해결에 필요한 요건 및 프롬프트 작성

### 2-1. 문제 해결에 더 필요한 요건 나열

문제 해결의 신속성과 전문성을 보장하기 위해, 기본 CoT 외에 **'역할 부여'** 및 **'제약 조건'** 유형을 조합하여 다음의 요건을 충족해야 합니다.

1.  **전문가 역할 부여 (Role-Playing):** 20년 경력의 전문가 페르소나를 명시하여 분석의 깊이와 통찰력을 확보합니다.
2.  **비교 분석 데이터 명시:** A, B, C 영상의 성과 지표를 명확히 제시하여 AI가 정확한 비교 진단을 수행하도록 유도합니다.
3.  **Chain of Thought (CoT) 5단계 강제:** 분석의 논리적 흐름을 구조화하여 원인 분석(진단)부터 대응 방안(처방)까지의 일관성을 확보합니다.
      * 1단계: 상황 및 핵심 문제 진단
      * 2단계: 데이터 비교 분석 및 가설 설정
      * 3단계: **3초 이탈률 80%** 원인 심층 분석
      * 4단계: 문제 진단 및 재정의 (핵심 인사이트 도출)
      * 5단계: 긴급 대응 방안 및 수정(Re-edit) 전략 제안
4.  **긴급 행동 지침 요구:** 보고서의 핵심 목적인 '즉각적인 수정(re-edit) 또는 대체 방안'을 구체적으로 요구합니다.
5.  **출력 형식 제약:** 보고서 형식(Markdown)을 지정하여 내일 오전 회의에 바로 활용 가능한 형태로 도출하도록 합니다.

### 2-2. 문제해결 프롬프트 작성 (역할 부여 + CoT + 제약 조건 조합)

```prompt
당신은 20년 경력의 프롬프트 엔지니어링 전문가이자 기업 리스크 해결 컨설턴트입니다. 현재 클라이언트 'A'(신규 뷰티 브랜드)의 광고 대행사 콘텐츠 제작 팀장 역할을 맡고 있습니다.

* **캠페인 대상:** 유튜브 범퍼 광고 (6초)

**[지시 사항: 5단계 Chain of Thought (CoT) 추론 강제]**
다음의 **5단계 CoT 과정**을 반드시 거쳐 전문적이고 논리적인 분석을 수행하고, 최종 보고서를 작성하십시오.

1.  **상황 및 핵심 문제 진단:** A/B 영상과의 비교를 통해 C 영상의 실패가 단순히 '할인' 문제가 아님을 지적하고, 핵심 데이터 지표(이탈률, CTR, 전환율)의 관계를 분석하여 클라이언트 리스크를 명확히 정의하십시오.
2.  **A/B/C 데이터 비교 분석 및 가설 설정:** C 영상의 '할인' 소구점과 A/B 영상의 '제품 본질' 소구점 차이에서 발견되는 소비자 행동 패턴의 차이(인지 단계의 문제인지, 구매 단계의 문제인지)에 대한 가설을 설정하십시오.
3.  **C 영상의 '3초 이탈률 80%' 원인 심층 분석:** 6초 범퍼 광고에서 3초 이탈률 80%가 의미하는 바를 분석하고, '할인 혜택 강조' 크리에이티브의 초반 3초 연출이 실패한 구체적인 원인 3가지(예: 소구점 노출 시점, 영상 구성, 톤앤매너)를 진단하십시오.
4.  **핵심 문제 진단 및 재정의 (인사이트 도출):** 앞선 분석을 종합하여, C 영상의 실패 원인을 '할인 혜택' 자체가 아닌, 'OOO' 때문에 발생한 문제로 재정의하고 명확한 인사이트를 제시하십시오.
5.  **긴급 대응 방안 및 수정(Re-edit) 전략 제안:** 클라이언트 리스크를 해소할 수 있는 2가지 긴급 대응 방안(1안: C 영상 즉시 Re-edit 전략, 2안: 즉각적인 대체 영상 제작 아이디어)을 구체적인 실행 계획과 함께 제안하십시오.

**[출력 형식 제약]**
결과를 **'## C 영상 긴급 대응 보고서 (컨텐츠 제작 팀장)'** 제목의 Markdown 파일 형식으로 제출하십시오.
```

### 2-3. 프롬프트 실행 결과 도출

## C 영상 긴급 대응 보고서 (컨텐츠 제작 팀장)

### 1\. 상황 및 핵심 문제 진단 (CoT 1단계)

| 지표 | A/B 영상 (제품 본질) | C 영상 (할인 혜택) | 진단 및 문제점 |
| :--- | :--- | :--- | :--- |
| **CTR** | 2.5% (순항) | **0.8% (치명적 저조)** | 초기 광고 노출 단계에서 타겟 소비자의 **관심 유발에 실패** |
| **구매 전환율** | 1.8% (양호) | **0.1% (사실상 전무)** | 광고 시청 후 구매 행동을 유도하는 **설득력이 없음** |
| **3초 이탈률** | N/A (상대적 양호) | **80% (매우 심각)** | 6초 영상의 **'초반 3초'가 광고 메시지 전달 자체를 실패**했음을 의미 |

**핵심 문제 정의:** C 영상의 실패는 단순한 마케팅 성과 저조를 넘어, 클라이언트의 **최대 광고 예산 손실 및 신뢰 관계 붕괴**라는 **기업 리스크**로 즉시 전환될 수 있는 심각한 상황입니다. 특히 3초 이탈률 80%는 C 영상의 메시지가 타겟에게 **도달조차 하지 못하고** 있다는 것을 명확히 보여줍니다.

-----

### 2\. A/B/C 데이터 비교 분석 및 가설 설정 (CoT 2단계)

| 영상 유형 | 소구점 | 성공/실패 이유 (추론) |
| :--- | :--- | :--- |
| **A/B 영상** | 성분/후기 (제품 본질) | 신규 브랜드임에도 **제품 자체의 효능과 가치**에 관심 있는 잠재 고객을 성공적으로 유인 (CTR/전환율 양호). **'탐색/인지' 단계**의 니즈 충족. |
| **C 영상** | 할인 혜택 | **'할인'만으로는** 신규 뷰티 브랜드에 대한 **신뢰와 흥미를 유발하지 못함**. 소비자들은 낯선 제품의 \*\*'할인 가격'보다 '제품의 가치'\*\*를 먼저 알고 싶어함. |

**설정 가설:** C 영상은 **신규 브랜드**라는 컨텍스트를 무시하고 **'구매' 단계** 소구점(할인)을 **'인지' 단계**에 섣불리 노출하여 실패했다. 소비자들은 "이 제품이 무엇인데 할인하는 거지?"라는 근본적인 의문을 해소하지 못하고 즉시 이탈한 것이다.

-----

### 3\. C 영상의 '3초 이탈률 80%' 원인 심층 분석 (CoT 3단계)

6초 범퍼 광고의 성패는 초반 1\~3초에 달려 있습니다. 80% 이탈은 광고의 **후킹(Hooking)** 전략이 완전히 실패했음을 의미하며, '할인 혜택 강조' 소구점의 구성적 오류는 다음과 같습니다.

1.  **소구점 노출 시점의 오류:** **초반 1\~3초**에 브랜드명/제품 설명 없이 \*\*'파격 할인\!'\*\*과 같은 텍스트만 등장했을 가능성이 높습니다. 신규 브랜드는 인지도가 낮으므로, '할인'이라는 정보는 **제품의 시각적 매력**이나 **궁금증을 유발하는 강력한 이미지** 뒤에 배치되어야 합니다.
2.  **시각적 매력 부재 (Generic Creative):** 할인 혜택을 강조하는 영상이 흔한 광고 템플릿처럼 보이거나, 뷰티 제품의 **고급스러움/트렌디함**을 놓쳤을 수 있습니다. 타겟 고객(뷰티 관심층)은 싼 가격보다 '예쁜 것'에 먼저 반응합니다.
3.  **메시지 과잉 압축 실패:** 6초 안에 '할인 정보'를 효과적으로 전달하려다 보니, 제품 자체의 시각적 소구점(제형, 발림성, 결과)을 놓치고 **단순 정보 나열**로 전락하여 광고적 흥미를 떨어뜨렸습니다.

-----

### 4\. 핵심 문제 진단 및 재정의 (인사이트 도출) (CoT 4단계)

**최종 진단 (인사이트):** C 영상의 실패 원인은 '할인 혜택' 자체가 아니라, \*\*'신규 브랜드의 제품 가치 인지(A/B의 성공 요소)'\*\*와 **'할인 정보'** 간의 **'순서 및 연결고리' 부재** 때문이다.

C 영상은 3초 만에 소비자를 낚으려 했으나, A/B 영상이 이미 입증했듯이, 이 클라이언트의 타겟은 \*\*'이 제품은 쓸만하다는 확신(성분/후기)'\*\*을 먼저 받은 후에야 \*\*'그래서 얼마인가(할인)'\*\*에 관심을 갖습니다. C 영상은 소비자의 \*\*'인지 허들'\*\*을 넘지 못하고 바로 \*\*'구매 허들'\*\*을 들이밀어 실패했습니다.

### 5\. 긴급 대응 방안 및 수정(Re-edit) 전략 제안 (CoT 5단계)

클라이언트의 신뢰 회복과 예산 손실 최소화를 위한 2가지 긴급 대응 방안을 제시합니다.

#### 1안: C 영상 즉시 Re-edit 전략 (가장 빠른 대응)

| 항목 | 기존 (실패 원인) | 수정 (Re-edit 전략) |
| :--- | :--- | :--- |
| **초반 3초 (Hooking)** | '할인' 텍스트/혜택 강조 | **A 영상의 '성분 강조' 후킹 이미지 재사용.** (예: 강력한 제품 제형, 신비로운 성분 이미지) |
| **중반 2초 (Bridge)** | 불필요한 제품 전경 노출 | **'A/B 영상의 성공 공식'을 가져와 제품의 매력(Benefit)을 빠르게 삽입.** |
| **후반 1초 (Call to Action)** | '0.1% 전환율' 실패의 근원 | **할인 혜택 정보를 후반 1초로 이동.** 혜택보다 '마감 임박'과 같은 희소성 메시지를 함께 전달. |
| **목표:** C 영상의 **CTR을 1.5% 이상**으로 회복시키고, 이탈률을 50% 이하로 낮춥니다. |

#### 2안: 즉각적인 대체 영상 제작 아이디어 (긴급 제작)

**전략:** A/B의 성공 요소('제품 본질')에 '할인'을 결합한 \*\*'성분 부각형 초단기 어필 영상'\*\*을 신규 제작합니다.

  * **제목:** "성분만 보고 3초 안에 결정하세요\!"
  * **크리에이티브 구성:**
    1.  **0\~2초 (제품 본질):** 'A 영상'에서 가장 반응이 좋았던 **핵심 성분 이미지와 효능 텍스트**를 압도적인 비주얼로 먼저 노출. (이탈률 방어)
    2.  **3\~4초 (신뢰 확보):** \*\*"A/B 테스트 검증 완료"\*\*라는 신뢰도 텍스트를 삽입하여, 성공한 두 영상의 레거시를 활용.
    3.  **5\~6초 (전환 유도):** \*\*"단 3일, 첫 구매 OOO원 혜택"\*\*처럼 기간 한정성을 강조한 할인 정보를 노출하여 **강력한 CTA**로 마무리.

**실행 계획:** A, B 영상에서 이미 검증된 푸티지(footage)를 활용하면, **오늘 오후 중으로 2안의 대체 영상 제작 및 심사 업로드**를 완료하고, C 영상 예산을 대체 영상에 즉시 투입할 수 있습니다. 클라이언트에게는 \*\*'실패 원인 분석 및 즉각적인 리소스 재활용 전략'\*\*으로 보고하여 리스크를 해소합니다.
