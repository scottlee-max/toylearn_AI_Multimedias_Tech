## ✅ 최적의 프롬프트 유형: **6. 체인 오브 스루트(Chain of Thought, CoT)**

---

### 🎯 선택 이유

| 구분            | 설명                                                                                               |
| ------------- | ------------------------------------------------------------------------------------------------ |
| **문제 성격**     | 단순한 사실 질의가 아니라, “왜 시청 지속 시간이 급감했는가?”와 “이를 개선하려면 어떤 조치를 취해야 하는가?”를 **논리적 단계**로 추론해야 하는 전략적 문제입니다. |
| **필요한 사고 과정** | - 데이터 분석적 사고 (무엇이 변했는가?)<br>- 인과 관계 추론 (왜 그렇게 되었는가?)<br>- 실행 전략 수립 (무엇을 해야 회복되는가?)               |
| **AI의 강점 활용** | CoT 프롬프트는 AI가 **단계별로 논리적 근거를 생성**하게 하여, 단순한 “감”이 아닌 **합리적 문제 해결 프로세스**를 제공합니다.                   |
| **결과물 기대 효과** | - 추론 기반 원인 분석<br>- 가설별 영향 요인 정리<br>- 실행 가능한 개선 전략 도출<br>- 데이터 기반 리포트 초안 자동화                      |

### 🧠 예시 프롬프트 (Chain of Thought 적용)

> “최근 1개월간 회사 유튜브 채널 신규 영상들의 평균 시청 지속 시간이 이전 분기 대비 30% 급감했습니다.
> 다음 단계를 따라 원인과 해결책을 도출해 주세요.
> 1️⃣ 가능한 원인 요인을 나열합니다.
> 2️⃣ 각 요인별 영향을 설명합니다.
> 3️⃣ 우선순위를 정해 해결 전략을 제시합니다.
> 4️⃣ 실행 가능한 액션 플랜을 제시합니다.
> 단계별로 논리적으로 생각하고, 데이터 기반의 마케팅 전문가처럼 답변해 주세요.”

| **조건부 생성형** | “‘시청 지속 시간 감소’ 원인과 ‘개선’ 키워드를 중심으로 작성해줘” | 주제 일관성 유지 가능하지만 분석 깊이는 제한됨 |
| **CoT형** ✅  | 문제 인식 → 요인 분석 → 해결 → 실행 단계로 구조화         | 복합 문제 해결에 가장 적합            |

## 🧠 유튜브 시청 지속 시간 감소 문제 분석 및 개선전략 보고서 자동 생성 프롬프트

```
[프롬프트 제목]
유튜브 채널 시청 지속 시간 급감 원인 분석 및 개선 전략 보고서 생성

[프롬프트 본문]
당신은 데이터 기반의 디지털 마케팅 전략 컨설턴트입니다.  
최근 1개월간 회사 유튜브 채널 신규 영상들의 평균 시청 지속 시간이 이전 분기 대비 30% 감소했습니다.  
다음 단계를 따라 **체계적 분석 보고서**를 작성하세요.

[단계별 지시]
1️⃣ 문제 정의
- 시청 지속 시간이 감소했다는 사실을 명확히 정의하고, 왜 중요한 KPI인지 설명합니다.
- 현재 데이터(예: 조회수, 클릭률, 평균 시청 시간, 이탈률, 업로드 주기 등)가 어떤 의미를 가지는지 해석합니다.

2️⃣ 원인 분석 (Chain of Thought)
- 가능한 모든 원인을 나열하고, 각 요인이 미친 영향을 단계적으로 추론합니다.
- 콘텐츠 측면(영상 길이, 주제, 썸네일, 오프닝 구성 등), 타깃 시청자 변화, 업로드 전략, 알고리즘 노출, 경쟁 콘텐츠 등으로 구분해 분석합니다.
- 각 원인의 근거를 논리적으로 제시합니다.

3️⃣ 데이터 인사이트
- 데이터로 추정 가능한 패턴이나 변화(예: 평균 영상 길이 증가 → 이탈률 상승)를 정리합니다.
- 가설별로 ‘검증 필요 지표’를 제시합니다.

4️⃣ 개선 전략 제안
- 상위 3개의 주요 원인에 대해 구체적 개선 전략을 제안합니다.
- 단기(1개월 이내), 중기(1~3개월), 장기(3개월 이상)로 구분합니다.
- 실행 우선순위를 정하고, 각 전략의 기대 효과를 수치나 예시로 제시합니다.

5️⃣ 실행 플랜
- 담당 부서, 주요 액션 아이템, 측정 지표(KPI)를 포함한 실행 계획표를 제안합니다.
- 개선 효과를 측정하기 위한 후속 데이터 분석 방법도 제안합니다.

[출력 형식]
보고서 형식으로 작성하되, 다음 구조를 따르세요:
1. 요약
2. 문제 정의
3. 원인 분석
4. 데이터 인사이트
5. 개선 전략
6. 실행 플랜
7. 결론 및 제언

[톤앤매너]
- 전문가 보고서 어조(객관적, 데이터 기반, 제안 중심)
- 불필요한 수식어 없이 명확하고 간결하게 작성
- 시각적 보고서 제작을 위한 구체적 항목 중심 구성
```
